# Contributing to NuraStore

Thank you for your interest in contributing to NuraStore! We welcome contributions from everyone, whether it's bug reports, feature requests, or code.

## How to Contribute

1.  **Fork the Repository:** Click the "Fork" button at the top right of this page.
2.  **Clone your Fork:** `git clone https://github.com/YOUR_USERNAME/nurastore.git`
3.  **Create a Branch:** `git checkout -b feature/AmazingFeature`
4.  **Make Changes:** Write your code.
5.  **Commit:** `git commit -m 'Add some AmazingFeature'`
6.  **Push:** `git push origin feature/AmazingFeature`
7.  **Open a Pull Request:** Go to the original repository and click "New Pull Request".

## Coding Standards

* **Python:** Follow PEP 8 style guidelines.
* **Commits:** Write clear, concise commit messages.
* **Testing:** Ensure your changes don't break existing functionality.

## Reporting Bugs

Please open an Issue on GitHub with:
1.  Description of the bug.
2.  Steps to reproduce.
3.  Expected vs. Actual behavior.

Thank you for helping make Nigerian E-commerce better!
